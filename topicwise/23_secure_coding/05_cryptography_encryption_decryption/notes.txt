Topic: cryptography Library

Goal:
Encrypt and decrypt sensitive data using the cryptography package.

Install:
pip install cryptography

Important:
Encryption is two-way.
You can decrypt encrypted data only if you have the correct key.

Use Cases:
- Encrypt sensitive files
- Encrypt database fields
- Encrypt tokens before storage
- Protect local configuration data

Run examples:
1. 01_generate_fernet_key.py
2. 02_encrypt_decrypt_text.py
3. 03_encrypt_decrypt_file.py
4. 04_wrong_key_decryption_failure.py
