Topic: hashlib

Goal:
Use hashlib for hashing text and checking file integrity.

Important:
Hashing is one-way. You cannot decrypt a hash.

Common Algorithms:
- SHA256
- SHA512
- MD5

Security Note:
Avoid MD5 and SHA1 for security-sensitive use cases.
Use SHA256 or stronger.
